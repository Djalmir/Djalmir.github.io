I'm Djalmir Miodutzki.

I build fast, secure, and scalable web apps using Vue.js and Node.js.
Since 2016, I’ve been developing complete web systems, always focusing on performance, clean architecture, and reliable user experiences.

On the front end, I develop SPAs and PWAs with Vue.js, JavaScript(ES6+) and Vite, using modern componentization patterns, reactive state management, and REST/JSON API integrations.
I also build interactive experiences with Canvas, WebSockets, and WebRTC.

More about me

On the back end, I work with Node.js and Express, implementing authentication, middlewares, and REST services.
I have strong experience with Linux-based infrastructure, including Ubuntu VPS, Nginx, PM2, SSL, continuous deployment, log analysis, optimization, and critical incident resolution.

I also develop integrations with local LLMs using Ollama, building secure, offline-first interfaces, automations, and pipelines for intelligent web applications.

I focus on clean code, consistent architectures, and long-term maintainability.
I see technical challenges as opportunities to design better systems — stable, scalable, and built to last.

I always strive to combine technical expertise, practical vision, and execution capabilities.
