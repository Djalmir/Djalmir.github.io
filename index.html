Tools

  • ZionJS

    ZionJS is a script that gives the developer the power of two way data binding and conditional rendering in pure javascript. In addition to facilitating the creation of http requests using the fetch API.

    It's not something I would recommend in a professional project yet. But I'm already able to work on my personal projects with it!

  • ZionCarousel

    ZionCarousel is an image carousel capable of reaching full screen mode with a simple click on any image in it. While in full screen, you can browse the images with the arrow keys.

    It is used many times on this site that you are now viewing.
